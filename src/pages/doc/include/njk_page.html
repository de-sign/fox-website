Pages

La tâche Gulp appellée html génère chaques pages *.html présentes dans un dossier qui lui est propre situé dans pages/.

Les informations présentes dans votre page sont restituées avec l'aide du fichier de configuration global data/base.json, étendue avec le fichier de configuration *.json propre à cette dernière si existant.

Pour créer votre propre page, il est recommandé de suivre les consignes suivantes :

Structure :
  • pages
    • my-page
      • my-page.html
      • my-page.json
my-page.html :
<!-- Extension du Layout -->
{% extends 'templates/' + FW.template.name + '.njk' %}

<!-- Extension des Blocks -->
{% block part %}
    <p>
        {{ FW.content.text | safe }}
    </p>
{% endblock %}
my-page.json :
{
    "url": {
        "path": "my-page.html"
    },

    "meta": {
        "title": "My Page",
        "description": "Un page statique"
    },

    "template": {
        "name": "my-template",
    },

    "content": {
        "text": "Ceci est le contenu de <b>part</b>"
    }
}

Si vous souhaitez modifier cette logique, n'oubliez pas de modifier le fichier gulp/config.js en conséquence !
