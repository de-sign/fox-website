Structure

Voici la structure proposé par FOX Website :

  • gulp
    • _build.js
    • _serves.js
    • config.js
  • src
    • assets
      • favicon
      • fonts
      • images
      • js
      • scss
        • abstracts
          • _mixins.scss
          • _variables.scss
        • base
          • _typography.scss
        • pages
      • video
    • data
      • website.json
    • pages
      • _templates
        • _head.njk
        • _macro.njk
  • gulpfiles.js

Celle ci est modulable mais n'oubliez pas de modifier le fichier de configuration gulp en conséquence !
