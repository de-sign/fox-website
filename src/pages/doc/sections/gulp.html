Gulp

A toolkit to automate
& enhance your workflow

FOX Website utilise Gulp afin d'automatiser les tâches de génération de votre site statique final : https://gulpjs.com

Afin de facilité le développement, FOX Website couple Gulp avec le module browser-sync vous permettant de visualiser en temps réel vos modifications : https://browsersync.io

Fonctionnement

Le fichier gulpfile.js inclus 3 fichiers : un fichier de configuration gulp/config.js, un fichier regroupant les tâches gulp/_builds.js et un fichier dédié aux services gulp/_serves.js.

Vous pouvez utiliser n'importe quel module gulp-* pour générer une nouvelle tâche, il faut pour cela ajouter cette dernière dans le fichier gulp/_builds.js et modifier le fichier de configuration gulp/config.js en conséquence !

Lignes de commandes

FOX Website mets à votre disposition plusieurs lignes de commandes : 

// Permet de visualiser votre site en temps réel
gulp

// -- Pour Windows
npm run fw-win-test
npm run fw-win-prod

// -- Pour UNIX
npm run fw-unix-test
npm run fw-unix-prod

// Permet de générer votre site
// -- Pour Windows
npm run fw-win-build-dev
npm run fw-win-build-test
npm run fw-win-build-prod

// -- Pour UNIX
npm run fw-unix-build-dev
npm run fw-unix-build-test
npm run fw-unix-build-prod

Libre à vous d'en ajouter dans le fichier packages.json !
