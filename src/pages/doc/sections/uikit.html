UIKit

A lightweight and modular front-end framework
for developing fast and powerful web interfaces

FOX Website utilise UIKit afin de définir un thème pour votre site : https://getuikit.com

Pour ce faire, FOX Website compile vos fichiers de style via la synthaxe SCSS en utilisant la structure de fichier proposée par Hugo Giraudel : https://sass-guidelin.es/fr/#architecture

La tâche Gulp appellée scss compile les fichiers présents dans assets/scss/.

Libre à vous d'utiliser un autre framework voir même de gérer vous-mêmes vos feuilles de style mais n'oubliez pas de modifier le fichier de configuration gulp en conséquence !
