Templates

La tâche Gulp appellée njk génère chaque templates présents dans un dossier qui lui est propre situé dans pages/_templates/.

Pour créer votre propre template, il est recommandé de suivre les consignes suivantes :

Structure :
  • pages
    • _templates
      • my-template
        • _part.njk
        • template.njk
      • _head.njk
template.njk :
<!doctype {{FW.doctype}}>
<html lang="{{FW.lang}}">
    <head>
        //=include ../_head.njk
    </head>
    <body>
        //=include _part.njk
    </body>
</html>
_part.njk :
{% block part %}
    // Contenu générique du bloc part
{% endblock %}

Dans ce cas, la tâche va générer un fichier my-template.njk dans le dossier pages/_templates/. C'est ce dernier que vous devrez utiliser comme template de référence pour vos pages statiques.

Si vous souhaitez modifier cette logique, n'oubliez pas de modifier le fichier gulp/config.js en conséquence !
