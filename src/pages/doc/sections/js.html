JavaScript

JavaScript is a programming language
that conforms to the ECMAScript specification

FOX Website utilise Javascript Vanilla afin de créer vos propres scripts : http://vanilla-js.com

La tâche Gulp appellée js unifie les fichiers présents dans un dossier qui lui est propre situé dans assets/js/.

Pour créer vos fichiers de scripts, il est recommandé de suivre les consignes suivantes :

Structure :
  • assets
    • js
      • my-script
        • _part.js
        • core.js
      • _require.js
core.js :
//=require ../require.js
//=include _part.js

window.eddEventListener(
    'load',
    my_function,
    false
);
_part.js :
function my_function() {
    console.log('my_function');
}

Libre à vous d'utiliser TypeScript ou de modifier cette logique mais n'oubliez pas de modifier le fichier de configuration gulp en conséquence !
