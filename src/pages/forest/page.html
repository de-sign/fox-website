Simple Node.js project for develop your static website

FOX Website utilise Gulp afin d'automatiser les tâches de génération de votre site statique final.
Afin de facilité le développement, ce dernier avec couplé avec le module browser-sync vous permettant de visualiser en temps réel vos modifications.

L'outils de templating Nunjucks est utilisé afin de générer vos pages statiques.
UIKit permet de définir un thème simple et efficace pour votre site.
Vos scripts intéractifs sont à créer avec Javascript Vanilla.
